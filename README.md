# 原神多账号辅助工具 v5.4

## 一、软件是什么

这是一个**帮你自动打原神每日任务**的工具。它会逐个登录你的原神账号，调用 BetterGI 完成每日委托、清体力等一条龙操作，一个号打完自动切下一个。你可以手动启动，也可以设好时间让它每天定点自动跑。

## 二、第一次使用 —— 必须配置的几项

### 2.1 准备工作

你需要先装好三样东西：

1. **原神** — 游戏客户端，确保能正常登录
2. **BetterGI** — 免费的原神自动化工具（请自行搜索下载），在 BetterGI 里配好一条龙任务
3. **本工具** — 解压到你喜欢的文件夹即可

另外，`tesseract-ocr` 文件夹必须跟 `GenshinMultiAccountTool.exe` 放在同一个目录下（压缩包里已经带好了）。

### 2.2 设置游戏路径

打开软件后，点击右上角的「设置」，会弹出一个设置窗口。按顺序填：

**1. BetterGI 路径（必填，两个都要选）**

- **BetterGI 可执行文件**：点「浏览」，找到你的 `BetterGI.exe`。
- **BetterGI 配置文件**：点「浏览」，找到 BetterGI 目录里的 `User\config.json`。这是 BetterGI 的一条龙配置，工具需要知道你已经配好了哪些任务。

**2. 原神路径（必填）**

- **原神可执行文件**：点「浏览」，找到你的 `YuanShen.exe`。国际服用户选 `GenshinImpact.exe`。
- **原神进程名**：一般不用改。默认填的是 `YuanShen.exe`，国际服用户手动改成 `GenshinImpact.exe`。这个名字是用来判断游戏有没有在运行的，写错就检测不到游戏。

填完后点「保存」。

> **重要提示**：游戏必须设置为窗口化运行。全屏模式下工具无法操作游戏画面。

### 2.3 添加账号

回到软件主界面，在左侧「账号列表」区域点「+ 添加账号」，会弹出填写窗口：

| 字段 | 说明 |
|------|------|
| **账号名称** | 给自己看的，比如「主号」「小号1」，随便写 |
| **启动方式** | 两个选项：**直接启动**（推荐，最常用，工具直接帮你启动原神然后调 BetterGI）；**胡桃启动**（需要装胡桃工具箱，通过胡桃来切号） |
| **BetterGI 一条龙配置** | 下拉列表会自动读取你在 BetterGI 里配好的任务，选一个即可 |
| **调度器配置组** | 一般不用管。当工具检测到当前 UID 和预期不符时，会自动执行这个配置组来纠正。可选 » |
| **胡桃中的账号名称** | 只有选了「胡桃启动」才需要填，填你在胡桃工具箱里给这个账号起的名字 |
| **游戏 UID** | 填这号对应的原神 UID（9 位数字）。工具用这个来验证当前登录的是不是对的号 |
| **任务完成后关闭软件** | 三个勾选框：打完这个号后要不要关掉游戏、关 BetterGI、关胡桃。默认全勾（推荐） |

填完点「保存」，账号就会出现在主界面列表里。重复这个步骤把你要跑的所有号都加进去。

### 2.4 其他设置说明

设置窗口里剩下的选项，按需调整即可，大部分保持默认就行：

**快捷键**

- **全局停止快捷键**（默认 Ctrl+Shift+Q）：任何时候按下，正在跑的号会被紧急停止。点旁边的「点击设置」再按你想用的组合键就能改。
- **全局暂停/继续快捷键**（默认 Ctrl+Shift+P）：暂停当前一条龙，再按一次恢复。

**开关选项**

- **启动时自动最小化窗口**：默认勾上。点了定时器的「一键启动全部」后，窗口会自动缩到托盘，不挡游戏画面。
- **关闭窗口时最小化到系统托盘**：默认勾上。点窗口右上角 X 不会退出程序，只会缩到右下角小图标。要完全退出请右键托盘图标选「退出」。
- **所有任务完成后自动关机**：默认不勾。勾上后全部号打完会有 60 秒倒计时，然后自动关机。倒计时期间可以取消。

**UID 识别方式**

这个决定工具怎么判断当前登录的是哪个号：

- **Tesseract OCR**：用 OCR 直接识别游戏画面里的 UID 数字。需要配好 Tesseract OCR 目录（压缩包自带，路径填 `tesseract-ocr` 文件夹即可）。优点是不依赖 BetterGI，缺点是必须窗口化、画面不能被遮挡。
- **BetterGI 调度组**（推荐）：让 BetterGI 内部直接读 UID。不需要 Tesseract，准确度更高，也不怕画面遮挡。选这个之后在下拉框里选一个 BetterGI 调度组即可。

**高级路径（一般不用动）**

- **胡桃工具箱路径**：仅当你用胡桃切号时才需要填。
- **胡桃 AppID**：仅 MSIX 版胡桃才需要，普通安装版不用管。
- **Tesseract OCR 目录**：选 Tesseract UID 方案时才需要，指向压缩包自带的那一整个 `tesseract-ocr` 文件夹。
- **一条龙超时时间**：默认 7200 秒（2 小时）。如果一个号的一条龙超过这个时间还没完成，工具会放弃这个号直接跳下一个。

## 三、常用功能介绍

### 启动/停止

在主界面勾选要跑的账号（左边列表里打勾），然后点「▶ 开始」。工具会从上到下逐个执行，当前在跑哪个号会高亮显示。

执行过程中可以用中间的「⏸ 暂停」和「■ 停止」按钮控制，也可以直接用全局快捷键。

### 多账号切换

工具会自动切换。流程是：
1. 启动原神（或通过胡桃切号）
2. 启动 BetterGI，读取一条龙配置
3. 等待 BetterGI 报到任务完成
4. 关闭相关软件，开始下一个号

勾选哪个就跑哪个，不需要人工干预。

### 定时计划

点击标题栏的「定时计划」打开计划管理窗口。

**添加一个定时任务**：选择执行模式（每天/每周/一次性/指定日期），选好时间，勾选要跑的账号，点「添加任务」。

快捷时间按钮（早8点/午12点/晚8点/凌晨4点）可以直接点，不需要手动调时间。

**启用所有任务**：点「一键启动全部」，定时器就开始运作了，到点自动触发一条龙。

**其他功能**：勾选「开机自启」可以让工具在电脑开机后自动运行；勾选「完成自动关机」会在全部任务结束后自动关机。

### 系统托盘

勾选「关闭窗口时最小化到系统托盘」后，点 X 不会退出程序，图标会缩到任务栏右下角。

右键点击托盘图标可以：开始/暂停/停止任务、打开设置、退出程序。双击图标可以打开主窗口。

## 四、常见问题

### 1. 提示"找不到 tesseract"或 UID 识别失败

**原因**：tesseract-ocr 文件夹没有跟 exe 放在一起，或者选了 Tesseract 方案但没配置路径。

**解决**：
- 确认 exe 旁边有 `tesseract-ocr` 文件夹，里面能看到 `tesseract.exe` 这个文件
- 到设置里重新选择 Tesseract OCR 目录
- 如果还不行，去设置里把「UID 识别方式」改成 BetterGI 调度组，这个方案完全不需要 Tesseract

### 2. 提示"程序已在运行中"

**原因**：工具已经开了一个，不能重复打开。

**解决**：看看任务栏右下角有没有工具的小图标，双击打开就行。找不到的话，按 `Ctrl+Shift+Esc` 打开任务管理器，搜 `GenshinMultiAccountTool`，结束进程后重开。

### 3. 检测不到游戏窗口 / 登录失败

**原因**：常见三种情况——游戏没开、开了但进程名写错了、或者游戏是全屏模式。

**解决**：
- 确认游戏是**窗口化**运行（在游戏设置里改成窗口模式）
- 到设置里检查「原神进程名」是否写对了（国际服是 `GenshinImpact.exe`，国服是 `YuanShen.exe`）
- 第一次用先只勾一个号试跑，如果出错，主界面右侧的日志区会显示详细的错误信息

### 4. 托盘图标不显示

**原因**：Windows 自动隐藏了，或者上次关闭时残留了旧图标。

**解决**：
- 点任务栏上的 `^` 小箭头，展开后看图标是不是在里面，按住拖到外面就行
- 如果展开也没有，打开任务管理器，结束掉所有 `GenshinMultiAccountTool.exe` 进程，然后重开
- 如果那个图标还在但是点了没反应（僵尸图标），重启一下 Windows 资源管理器就行：任务管理器里找到「Windows 资源管理器」，右键点重新启动

### 5. 定时任务到点不执行

**原因**：定时器没开启，或者电脑休眠了。

**解决**：
- 打开定时计划窗口，确认顶部状态是「运行中」，任务列表的勾都打上了
- 电脑**绝对不能休眠或睡眠**（去系统设置 → 电源 → 把休眠关掉，改成"永不"）
- 可以先手动点「一键启动全部」试一次，看能不能正常执行
- 想无人值守的话：勾上「开机自启」，电源设「永不睡眠」，定时计划保持运行就行
