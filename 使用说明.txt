原神多账号辅助工具 v5.4
==========================

一、软件是什么
-------------
这是一个帮你自动打原神每日任务的工具。它会逐个登录你的原神账号，调用 BetterGI 完成每日委托、清体力等一条龙操作，一个号打完自动切下一个。你可以手动启动，也可以设好时间让它每天定点自动跑。

二、第一次使用 —— 必须配置的几项
--------------------------------
2.1 准备工作

你需要先装好三样东西：
1. 原神 — 游戏客户端，确保能正常登录
2. BetterGI — 免费的原神自动化工具（请自行搜索下载），在 BetterGI 里配好一条龙任务
3. 本工具 — 解压到你喜欢的文件夹即可

另外，tesseract-ocr 文件夹必须跟 GenshinMultiAccountTool.exe 放在同一个目录下（压缩包里已经带好了）。

2.2 设置游戏路径

打开软件后，点击右上角的「设置」，会弹出一个设置窗口。按顺序填：

【BetterGI 路径（必填，两个都要选）】
- BetterGI 可执行文件：点「浏览」，找到你的 BetterGI.exe。
- BetterGI 配置文件：点「浏览」，找到 BetterGI 目录里的 User\config.json。这是 BetterGI 的一条龙配置，工具需要知道你已经配好了哪些任务。

【原神路径（必填）】
- 原神可执行文件：点「浏览」，找到你的 YuanShen.exe。国际服用户选 GenshinImpact.exe。
- 原神进程名：一般不用改。默认填的是 YuanShen.exe，国际服用户手动改成 GenshinImpact.exe。这个名字是用来判断游戏有没有在运行的，写错就检测不到游戏。

填完后点「保存」。

重要提示：游戏必须设置为窗口化运行。全屏模式下工具无法操作游戏画面。

2.3 添加账号

回到软件主界面，在左侧「账号列表」区域点「+ 添加账号」，会弹出填写窗口：

┌──────────────┬──────────────────────────────────────────────────┐
│ 字段           │ 说明                                             │
├──────────────┼──────────────────────────────────────────────────┤
│ 账号名称       │ 给自己看的，比如「主号」「小号1」，随便写          │
│ 启动方式       │ 直接启动（推荐）：工具直接帮你启动原神调 BetterGI │
│                │ 胡桃启动：需要装胡桃工具箱，通过胡桃来切号        │
│ BetterGI 一条龙│ 下拉列表自动读取你在 BetterGI 里配好的任务         │
│ 配置           │ 选一个即可                                      │
│ 调度器配置组   │ 一般不用管。UID 不符时自动纠正用，可选            │
│ 胡桃中的账号名 │ 只有「胡桃启动」才需要填                         │
│ 游戏 UID       │ 9 位数字，工具用来验证登录的是不是对的号          │
│ 任务完成后     │ 三个勾选框：关游戏 / 关 BetterGI / 关胡桃         │
│ 关闭软件       │ 默认全勾（推荐）                                 │
└──────────────┴──────────────────────────────────────────────────┘

填完点「保存」，账号就会出现在主界面列表里。重复这个步骤把你要跑的所有号都加进去。

2.4 其他设置说明

【快捷键】
- 全局停止（默认 Ctrl+Shift+Q）：紧急停止当前任务。点「点击设置」再按组合键即可修改。
- 全局暂停/继续（默认 Ctrl+Shift+P）：暂停一条龙，再按恢复。

【开关选项】
- 启动时自动最小化窗口：默认勾。定时器启动后窗口自动缩到托盘，不挡游戏画面。
- 关闭窗口时最小化到系统托盘：默认勾。点 X 不会退出，缩到右下角图标。完全退出需右键托盘图标选「退出」。
- 所有任务完成后自动关机：默认不勾。勾上后全部号打完有 60 秒倒计时自动关机，期间可取消。

【UID 识别方式】
决定工具怎么判断当前是哪个号：
- Tesseract OCR：识别游戏画面里的 UID 数字。需要配 Tesseract OCR 目录（压缩包自带）。优点不依赖 BetterGI，缺点必须窗口化、画面不能被遮挡。
- BetterGI 调度组（推荐）：BetterGI 内部直接读 UID。不需要 Tesseract，准确度更高。选这个后在下拉框里选调度组。

【高级路径（一般不用动）】
- 胡桃工具箱路径：仅胡桃切号时需要。
- 胡桃 AppID：仅 MSIX 版胡桃需要。
- Tesseract OCR 目录：选 Tesseract UID 方案时需要。
- 一条龙超时时间：默认 7200 秒（2 小时）。超时未完成会跳过当前号。

三、常用功能介绍
----------------
【启动/停止】
在主界面勾选要跑的账号，点「▶ 开始」。从上到下逐个执行，当前在跑的号会高亮。执行中可用「⏸ 暂停」和「■ 停止」或全局快捷键控制。

【多账号切换】
工具自动切换，流程：启动原神 → 启动 BetterGI → 等任务完成 → 关闭软件 → 下一个号。

【定时计划】
点标题栏「定时计划」打开：
- 添加任务：选模式（每天/每周/一次性/指定日期），选时间，勾账号，点添加。
- 快捷时间：早8点 / 午12点 / 晚8点 / 凌晨4点，一键设定。
- 一键启动全部：开启定时器，到点自动跑。
- 开机自启：电脑开机后工具自动运行。
- 完成自动关机：全部任务结束后自动关机。

【系统托盘】
点 X 缩到右下角。右键托盘图标：开始/暂停/停止任务、打开设置、退出程序。双击打开主窗口。

四、常见问题
------------
1. 提示"找不到 tesseract"或 UID 识别失败
   解决：确认 exe 旁边有 tesseract-ocr 文件夹；到设置重新选 Tesseract OCR 目录；
   或者直接把 UID 识别方式改成 BetterGI 调度组（不需要 Tesseract）。

2. 提示"程序已在运行中"
   解决：看任务栏右下角有没有工具图标，双击打开。找不到就按 Ctrl+Shift+Esc
   进任务管理器，搜 GenshinMultiAccountTool，结束进程后重开。

3. 检测不到游戏窗口 / 登录失败
   解决：确认游戏是窗口化运行；确认进程名写对（国服 YuanShen.exe，国际服
   GenshinImpact.exe）；先只勾一个号试跑，看右侧日志区的错误信息。

4. 托盘图标不显示
   解决：点任务栏 ^ 小箭头展开，看图标是不是在里面；打开任务管理器结束所有
   GenshinMultiAccountTool.exe 后重开；如果图标是僵尸状态，打开任务管理器
   找到「Windows 资源管理器」右键点「重新启动」。

5. 定时任务到点不执行
   解决：确认定时计划窗口顶部状态是「运行中」，任务列表勾都打上；电脑绝对
   不能休眠或睡眠（去系统设置 → 电源把休眠关掉，改成"永不"）；先手动点
   「一键启动全部」试一次；想无人值守：勾开机自启 + 永不睡眠 + 定时计划保持运行。
